
# [Rubinho][rubinho]

**[Rubinho][rubinho]** is a [RubyGems][rubygems] visualization tool.

The graph visualization is made using [VivaGraphJS][vivagraph].

Check out [yasiv][yasiv] for more inspiration and the [NPM][npm] visualization tool.
